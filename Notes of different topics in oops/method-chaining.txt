Method Chaining in PHP
1. Simple English Definition: Method Chaining is a technique that allows calling multiple methods of a class in a single statement, one after another, using the arrow operator (->)
. It involves linking methods into a "chain" where each method acts on the result or instance of the previous one
.
2. Hinglish Explanation (Asaan Bhasha Mein):
Jab hum ek class ka object banate hain aur uske saath ek hi line mein kayi saare functions (methods) ko call karte hain, toh use Method Chaining kehte hain
.
Iska sabse bada fayda ye hai ki humein har baar naya object variable likhne ki zaroorat nahi padti. Ek hi object par hum poori chain chala sakte hain
.
Laravel jaise bade frameworks mein iska upyog "Fluent Strings" ke liye bhi hota hai taaki string par kayi operations (jaise uppercase karna, trim karna) ek saath kiye ja sakein
.
Method Chaining Ka Sabse Zaroori Rule
Bhai, method chaining tabhi kaam karegi jab aapka har method aakhir mein $this keyword ko return kare
.
$this ka matlab hota hai "current class ka object"
.
Agar aap kisi method se $this return nahi karoge, toh PHP ko agli call ke liye object nahi milega aur woh Fatal Error de dega
.
Practical Code Example (Copy karne ke liye):
<?php
class Company {
    public function getName() {
        echo "This is Honda Motors. ";
        // Agle method ko call karne ke liye khud ko return karna zaroori hai
        return $this; [3]
    }

    public function getEmployees() {
        echo "It has 30,000 employees. ";
        return $this; [5]
    }

    public function getOffice() {
        echo "Total offices: 200.";
        // Aakhri method mein return $this optional hai
    }
}

// Object banana
$obj = new Company();

// Normal tarika (Bina chaining ke):
// $obj->getName();
// $obj->getEmployees();

// Method Chaining wala tarika:
$obj->getName()->getEmployees()->getOffice(); [1, 3]
?>
Mukhya Baatein (Key Points for Notes):
Readability: Isse code "children wala" nahi lagta aur ek sequence mein dikhta hai
.
Return Requirement: Chaining tab tak chalti rahegi jab tak methods $this return karte rahenge
.
Fluent Interface: Is technique ka upyog karke hum fluent interfaces banate hain jo padhne mein natural language (English) jaisa lagta hai
.
Use Case: Iska sabse zyada istemal Database Queries (Laravel Eloquent) aur String manipulation mein hota hai
.
