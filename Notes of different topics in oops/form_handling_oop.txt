Call Class Function with Form in PHP
1. Simple English Definition: Calling a class function with a form involves triggering a specific method of a PHP object when a user submits an HTML form. This is typically achieved by checking for a request (POST or GET) and then instantiating the class to call its method.
2. Hinglish Explanation (Asaan Bhasha Mein):
Jab hum OOP use karte hain, toh hum chahte hain ki user jab form submit kare, tab hamari class ka koi khaas function (method) chale.
PHP mein directly <button onclick="class->method()"> nahi likh sakte kyunki PHP server-side chalti hai aur HTML client-side.
Isliye humein pehle check karna padta hai ki kya form submit hua hai (isset() ka use karke), aur phir class ka Object banakar uske method ko call karte hain.
Implementation Steps (Kaise karein):
HTML Form banao: Ek form create karein jisme method="post" ho.
Submit Check karo: PHP mein isset($_POST['button_name']) ka upyog karke check karein ki button dabaya gaya hai ya nahi.
Class aur Method define karo: Class ke andar woh logic likhein jo aap button click par chalana chahte hain.
Object banakar call karein: Agar submit set hai, toh class ka naya instance (object) banayein aur arrow operator (->) se function call karein.
Practical Code Example (Copy karne ke liye):
<?php
// 1. Check karna ki kya form submit hua hai
if (isset($_POST['user_name'])) {
    $inputData = $_POST['user_name'];

    // 2. Class definition
    class UserProfile {
        function showMessage($name) {
            echo "<h3>User Name is: " . $name . "</h3>";
            echo "Class function successfully called via Form!";
        }
    }

    // 3. Object banana aur function call karna
    $userObj = new UserProfile();
    $userObj->showMessage($inputData);
}
?>

<!-- 4. HTML Form -->
<form method="post" action="">
    <input type="text" name="user_name" placeholder="Enter User Name" required>
    <br><br>
    <button type="submit">Call Class Function</button>
</form>
Mukhya Baatein (Key Points for Notes):
Action Property: Agar action ko khali choda jaye, toh request usi same page par aati hai jahan form likha hai.
Data Passing: Form se aaya data (jaise $_POST['user']) hum class function ke parameters mein pass kar sakte hain taaki logic dynamic rahe.
Security: Hamesha check karein ki data present hai (isset) warna PHP "Undefined Index" ka error de sakta hai.
Hiding Form: Aap else condition ka use karke form ko chhupa sakte hain jab result display ho jaye.
