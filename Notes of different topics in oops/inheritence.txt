Inheritance in PHP
1. Simple English Definition: Inheritance is an OOP principle that allows a sub-class (child) to inherit the properties and methods of a super-class (parent). It enables code reuse and establishes a relationship between different classes
.
2. Hinglish Explanation (Asaan Bhasha Mein):
Inheritance ka matlab hai "virasat". Jaise ek bachhe ko apne maa-baap se gun (traits) aur jaydad milti hai, waise hi programming mein ek class doosri class se uske functions aur variables le sakti hai
.
Iske liye PHP mein extends keyword ka upyog kiya jata hai
.
Jo class apni khoobiyan deti hai use Parent ya Super Class kehte hain, aur jo unhe leti hai use Child ya Sub Class kehte hain
.
Mukhya Baatein (Key Points for Notes):
"Is-A" Relationship: Inheritance hamesha tab lagaya jata hai jab do classes ke beech "Is-A" ka rishta ho (Jaise: Car is a Vehicle)
.
Method Overriding: Agar parent aur child dono mein same naam ka method hai, toh child wala method parent wale method ko override (daba) deta hai
.
Single Inheritance: PHP mein ek class sirf ek hi parent class ko extend kar sakti hai
.
Access Control: Protected modifier khaas taur par inheritance ke liye bana hai, jisse child class toh data use kar sakti hai par bahar ka koi nahi
. Private cheezein inherit nahi hoti hain
.
Final Keyword: Agar aap chahte ho ki koi aapki class ko inherit na kar sake, toh uske aage final laga do
.
Practical Code Example (Copy karne ke liye):
<?php
// 1. Parent Class (Super Class)
class Vehicle {
    public $wheels;

    public function start() {
        echo "Vehicle is starting...";
    }
}

// 2. Child Class (Sub Class)
class Car extends Vehicle {
    // Isne Vehicle ki saari properties le li hain
    public function setWheels() {
        $this->wheels = 4; // Wheels property parent se aayi hai
        echo "Car has " . $this->wheels . " wheels.";
    }
}

$myCar = new Car();
$myCar->start(); // Output: Vehicle is starting... [Parent method call]
$myCar->setWheels(); // Output: Car has 4 wheels.
?>
Bhai, dhyan rakhna: Agar tujhe multiple inheritance ki kami mehsoos ho (kyunki PHP sirf ek parent allow karta hai), toh tu Traits ka use kar sakta hai
.
